# Operating system icon sources

These SVG files are bundled locally so the application does not fetch remote assets at runtime.

- `debian.svg`: Debian Open Use Logo without wordmark, downloaded from
  <https://www.debian.org/logos/openlogo-nd.svg>. Debian publishes it under
  LGPL-3.0-or-later or CC BY-SA 3.0.
- `amazon.svg`, `oracle.svg`, `windows.svg`: Devicon v2.17.0 assets, downloaded
  from <https://github.com/devicons/devicon>. Devicon is MIT licensed; the
  depicted names and logos remain trademarks of their respective owners.
- All other SVG files: Simple Icons v16.21.0 assets, downloaded from the
  versioned `simple-icons` package on jsDelivr. Simple Icons is CC0-1.0; the
  depicted names and logos remain trademarks of their respective owners.

The icons are used only to identify the operating system detected on a saved
SSH host. No endorsement or affiliation is implied.
